# spring-gift-product

# 1단계

## 구현할 기능

* 상품 정보를 담기 위한 Product Class 구현
* 상품 전체 조회 API
* 상품 단건 조회 API
* 상품 추가 API
* 상품 수정 API
* 상품 삭제 API

# 2단계

## 구현할 기능

* [Controller]
    * 상품 관리 페이지 응답용 컨트롤러 구현
* [View]
    * 상품 전체 조회 화면
    * 상품 추가 버튼 구현
    * 상품 수정 버튼 구현
    * 상품 삭제 버튼 구현

# 3단계

## 구현할 기능

* h2 데이터베이스 연동
* 상품 정보를 저장하기 위한 ProductRepository 구현
