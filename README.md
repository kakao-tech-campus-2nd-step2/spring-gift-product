# spring-gift-product
step1 구현할 기능 목록
1. Product record
2. Product Controller
   - 상품 추가
   - 상품 전체 조회
   - 상품 ID로 검색해서 조회
   - 상품 ID로 검색해서 수정
   - 상품 ID로 검색해서 삭제

step2 구현할 기능 목록
1. 관리자 화면 메인 페이지(상품 리스트)
2. 관리자 화면 상품 상제 페이지
3. 관리자 화면 상품 추가 및 수정 페이지
4. Product Controller
   - 상품 추가 + 뷰 매핑
   - 상품 전체 조회 + 뷰 매핑
   - 상품 ID로 검색해서 조회
   - 상품 ID로 검색해서 수정
   - 상품 ID로 검색해서 삭제
   - 상품 수정 뷰 매핑
   - 상품 추가 뷰 매핑
